布局样文

正文始终停留在视线中心

窗口变宽时，WorldForge 增加上下文空间与安全留白，而不是无限拉长正文。版心以 CSS 像素计量，在不同系统缩放与显示器之间保持稳定的阅读节奏。

界面缩放只改变控件、侧栏和 UI 字体；正文字号与正文宽度由各自设置控制。跨屏后重新判断断点，窗口坐标则以 DIP 保存并校验可见区域。

这是一段用于截图与边界测试的公开布局样文，不会写入作者项目或应用数据库的正文区域。

当有效视口低于 1100 像素，右侧设置进入抽屉；低于 900 像素时，左右两侧都成为可由键盘关闭的抽屉。整页保持无水平滚动，浮层与对话框也会重新定位到可见范围内。
