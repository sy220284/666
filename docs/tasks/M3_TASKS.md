# M3 任务卡：AI生成闭环

## M3-01 Provider、连接测试与凭据

- 目标：安全连接外部API和用户已运行的本地兼容服务。
- 依赖：M0-06、M2。
- 分支：`feat/m3-provider-layer`
- 关联：REQ-023、024、043，P0-022、067、070。

### 实现

- `AIProvider`最小接口。
- OpenAI兼容和Anthropic适配器。
- 连接测试：地址、认证、最短生成、流式和结构化输出。
- OS Credential Store；数据库只保存`credentialRef`。
- 本机、局域网和外部端点提示。
- 标准错误映射和超时设置。
- ModelSupportProfile基础结构。

### 非目标

不安装或监管本地模型；不建设任意脚本式Custom插件；不增加推测性能力位。

### 测试

正常、认证失败、限流、超时、中断、取消、缺Token统计、无模型列表和本地无密钥服务。

## M3-02 约束包与FTS5检索

- 目标：为每类AI任务提供可追溯、可裁剪的本章记忆包。
- 依赖：M2-04、M3-01。
- 分支：`feat/m3-constraint-package`
- 关联：REQ-025，P0-025、026相关Eval。

### 实现

- P0代码约束、P1章节必须项、P2设定状态、P3文风声音、P4辅助背景。
- 读取当前章、SceneBeat、前章尾快照、实体当前状态、知情、伏笔和作品规则。
- FTS5补充召回。
- 当前/历史状态时序过滤。
- 去重、冲突标记、Token估算、安全边距和裁剪日志。
- `contentHash`、来源ID和版本ID。

### 非目标

V1不实现Embedding、向量库、Rerank和通用检索Adapter。

### 测试

必选项不被P4挤出；历史状态不会冒充当前状态；过期快照不进入约束包；同一输入Hash稳定；Token超限返回明确错误。

## M3-03 T0/T1、快速改写、融合与取消

- 目标：完成可选、可取消、可降级的AI生成工作流。
- 依赖：M3-01、M3-02、M0-04。
- 分支：`feat/m3-generation-workflows`
- 关联：REQ-026—028，P0-023—028。

### 实现

- GenerationRun和requestId幂等。
- T0结构化骨架多候选。
- T1基于选定骨架生成完整Candidate。
- 快速改写：单段内联预览、换一个、取消和应用后撤销。
- 结构性改写：跨段、跨场景、整章进入完整Candidate。
- 融合：BeatSourceMapping和过渡修补。
- 真实stage事件、字符数、耗时、取消和partial Candidate。

### 安全

- 流式文本不直接写Draft。
- 快速改写同样执行LockGuard、Revision和来源标记。
- 无法可靠判断轻量范围时升级结构性流程。

### 测试

切章续跑、多任务并行、取消、断流、无效JSON、骨架遗漏、部分候选和重启后Run/Candidate状态。

## M3-04 候选Diff、冲突、采用与回退

- 目标：让作者在长章节中高效比较并安全采用候选。
- 依赖：M1-04、M3-03、M0-06。
- 分支：`feat/m3-candidate-review`
- 关联：REQ-013、029，P0-029—032。

### 实现

- `logicalBlockId`结构Diff优先。
- 拆分、合并、新增、删除识别。
- 块内中文字符Diff。
- 双栏、上下、单稿、只看差异、折叠未改段和场景导航数据接口。
- 同步滚动与解除同步。
- 选择映射生成Block Patch。
- Revision/Hash/锁定冲突集。
- 原子采用、ApplyRecord、即时撤销和重启后回退。

### 性能

5000字首屏≤500ms；完整Diff≤1.2s；可取消；不阻塞编辑器输入。

### 测试

同一段同时修改、块被删除、候选拆分段落、锁定块、部分候选、全选/局部选、撤销和冲突后二次提交。
