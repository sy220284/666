# 已知风险

- 当前环境没有两台物理显示器或真实热插拔设备；显示器丢失与混合 DPI 恢复由确定性显示快照覆盖，原生 scale factor 由单个 Xvfb Electron 进程验证。
- Windows、macOS、签名安装包与系统级跨屏行为仍由 M8 发布矩阵验收。
- “安静编辑部”仅是 M0-06 显示技术原型，最终业务视觉、双主题与无障碍完整验收属于 M7。
- 150% 原生缩放截图因 Chromium 的分数 DIP 舍入为 2562x1440；有效视口为 1708x960，与名义 1707 CSS px 相差 1px，测试允许最多 2px 且确认无内容裁切。
- M1 应用设置必须复用 app.sqlite.window_preferences 与现有 IPC 契约，不得另建窗口状态旁路。
