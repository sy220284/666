| ID | 验收项 | 通过标准 | 证据 |
|---|---|---|---|

## 10. 发布判断

- 任一代码硬保证失败：禁止发布。
- 任一P0数据安全或恢复用例失败：禁止发布。
- 模型质量未达标：对应AI任务降级，但只要核心自主写作闭环可用，不必阻止整个应用发布。
- 所有通过项必须在追踪矩阵中链接任务和证据目录。
