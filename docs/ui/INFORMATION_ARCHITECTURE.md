# WorldForge V1.0 信息架构与导航

> 状态：Frozen  
> 目标：控制页面数量，确保人物弧光、节奏、候选和恢复都有明确入口。

## 1. 一级导航

V1.0固定六个一级入口：

```text
首页
规划
写作
设定
检查
设置
```

导入导出、备份恢复和搜索属于跨模块操作，不单独占用一级导航。

## 2. 页面地图

```text
首页
├─ 最近项目
├─ 继续写作
├─ 新建/导入
├─ 项目健康状态
└─ 运行任务与恢复入口

规划
├─ ProjectBrief
├─ 大纲树
├─ 卷与章节
└─ SceneBeat

写作
├─ 章节编辑器
├─ 候选审阅
├─ 冲突处理
├─ 历史版本
└─ 沉浸写作

设定
├─ 人物/地点/势力/道具/能力/规则/事件
├─ Canon与当前/历史状态
├─ 人物弧光与里程碑
├─ 时间线
├─ 知情信息
└─ 伏笔

检查
├─ 本章检查
├─ 全书问题
├─ 人物弧光一致性
├─ 网文节奏与黄金三章
├─ 搜索与替换
├─ 项目词典
├─ StoryTodo与Comment
├─ 导入导出
└─ 备份恢复

设置
├─ 通用
├─ 编辑器
├─ 外观与显示
├─ AI连接
├─ 备份
├─ 检查、节奏与词典
└─ 高级
```

“故事指南针”不作为独立功能或页面名称；相关内容分别归入ProjectBrief、大纲和SceneBeat。

## 3. 三个核心工作台

### 3.1 规划工作台

```text
左：大纲树与卷章目录
中：ProjectBrief/节点详情/SceneBeat
右：相关人物、设定、伏笔、弧光和目标摘要
```

规划变化不自动修改正文。

### 3.2 写作工作台

```text
左：卷章目录
中：正文编辑器
右：本章 / AI / 人物 / 设定 / 检查
底：保存、字数、任务和检查状态
```

人物标签可显示当前状态与已确认弧光阶段；检查标签可显示节奏建议摘要。

### 3.3 检查与交付工作台

```text
左：问题、节奏或交付分类
中：问题列表/搜索结果/预览
右：证据、建议、影响和操作
```

## 4. 导航层级

1. 一级导航：切换工作领域。
2. 工作台目录：选择卷、章、实体、弧光或问题分类。
3. 标签：切换同一对象视图。
4. 抽屉/弹层：处理短时任务。

同一操作不得在一级导航、侧栏和顶部菜单重复出现三次以上。

## 5. 首页

### 新手默认

- 继续写当前章节。
- 最多一条下一步建议。
- 最近项目。
- 新建和导入入口。

### 专业默认

- 最近项目列表。
- 最近章节、字数和修改时间。
- 运行任务和数据风险。
- 快速创建空白项目。

首页主动提示最多2条，保存失败和数据库损坏优先。

## 6. 项目顶部栏

固定：项目名、一级入口、全局搜索、运行任务、保存/健康状态、帮助。  
不固定：大字数卡、多个模型状态灯、全部校验数量和复杂面包屑。

## 7. 左侧目录

- 支持折叠、调宽、键盘和虚拟列表。
- 显示卷章状态、字数、任务提示和回收站。
- 拖动时显示目标位置与跨章影响。

## 8. 右侧上下文栏

```text
本章 | AI | 人物 | 设定 | 检查
```

- 本章：目标、SceneBeat、字数、状态、尾快照和定稿。
- AI：任务、模型支持、约束摘要和候选历史。
- 人物：当前人物、状态、知情、声音样本和已确认弧光阶段。
- 设定：地点、规则、物品、势力和伏笔。
- 检查：问题、弧光冲突、节奏建议和正文锚点。

完整编辑跳转到设定或检查工作台。

## 9. 全局搜索

范围：当前章、活动正文、历史Version和实体。V1.0 P0不搜索研究笔记或附件。

默认简单搜索；正则、大小写和组合范围只在高级入口展示。

## 10. 弹层、抽屉与独立页面

弹层：简短确认、单对象快速编辑、少量参数。  
抽屉：窄屏上下文、AI参数、选区操作、搜索预览。  
独立全屏页：候选比较、冲突处理、导入预览、备份恢复和损坏诊断。

高级三栏冲突视图属于P1；P0默认使用当前稿/候选稿双栏与结果摘要。

## 11. 新手与专业模式

| 区域 | 新手模式 | 专业模式 |
|---|---|---|
| 规划 | 问题式少字段 | 完整树和字段 |
| 右侧栏 | 摘要和推荐动作 | 来源、规则和审计信息 |
| AI | 续写/改写等任务语言 | Run、Prompt和支持档案 |
| 检查 | 高风险与通俗建议 | 规则、证据、误报和批处理 |

模式切换不改变数据、命令和恢复能力。

## 12. 页面返回

- 从问题、节奏建议或弧光节点跳正文后，返回原列表和滚动位置。
- 从人物进入弧光详情，返回写作时恢复选区。
- 候选关闭后回到发起章节和原滚动位置。
- 任务运行中切页，任务条保持可见。
- 重启恢复最近项目、章节、编辑位置和侧栏状态；不恢复临时对话框。

## 13. 禁止结构

- 不增加AI中心、数据中心、运营中心或插件中心。
- 不把所有功能塞进右侧栏。
- 不让设置页承担正文或候选业务。
- 不创建新手版与专业版两套页面代码。
- 不为P1/V1.5功能展示可用入口。
