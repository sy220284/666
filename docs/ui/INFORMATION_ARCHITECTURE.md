# WorldForge 信息架构与导航

> 状态：Frozen  
> 目标：控制页面数量、入口层级和工作台切换，避免把写作工具做成复杂后台。

## 1. 一级导航

V1.0固定六个一级入口：

```text
首页
规划
写作
设定
检查
设置
```

“导入导出”“备份恢复”“搜索”属于跨模块操作，不单独占用一级导航。

## 3. 三个核心工作台

### 3.1 规划工作台

结构：

```text
左：大纲树/卷章目录
中：当前节点详情与场景列表
右：人物、设定、伏笔、目标摘要
```

用途：整理结构、拖动顺序、建立SceneBeat和关联实体。

### 3.2 写作工作台

结构：

```text
左：卷章目录
中：正文编辑器
右：本章 / AI / 人物 / 设定 / 检查
底：保存、字数、AI任务与校验状态
```

用途：完成日常写作、快速改写、候选审阅、定稿和状态确认。

### 3.3 检查与交付工作台

结构：

```text
左：问题分类或交付步骤
中：问题列表/搜索结果/预览
右：证据、建议、影响与操作
```

用途：修订、全书搜索、导入导出、备份与恢复。

## 4. 导航层级

导航优先级：

1. 一级导航：切换工作领域。
2. 工作台目录：选择卷、章、实体或问题分类。
3. 标签：切换同一对象的不同视图。
4. 抽屉/弹层：处理短时任务。

同一操作不得同时在一级导航、右侧栏和顶部菜单重复出现三次以上。

## 5. 首页设计

首页只承载启动任务，不承载完整管理后台。

### 新手默认

- “继续写第X章”主入口。
- 建书进度或轻量下一步建议。
- 最近一个项目。
- 创建新项目和导入作品。

### 专业默认

- 最近项目列表。
- 每个项目最近章节、字数、最后修改时间。
- 运行中任务与数据风险。
- 快速新建空白项目。

### 首页状态限制

- 主动提示最多2条。
- P0保存失败、数据库损坏等数据风险优先。
- 低风险校验和一般备份提醒只在对应工作台展示。

## 6. 项目内顶部栏

固定内容：

```text
当前项目名称
当前一级入口
全局搜索
运行任务入口
保存/数据健康状态
窗口与帮助入口
```

不固定展示：

- 今日字数大卡片。
- 多个AI模型状态灯。
- 全部校验数量。
- 复杂面包屑。

当路径超过“项目 > 卷 > 章”时，使用简短面包屑或当前对象标题。

## 7. 左侧目录

- 支持折叠、宽度调整和键盘导航。
- 卷章树使用虚拟列表。
- 当前章节、未完成、定稿、回收站状态通过图标和文字表达。
- 目录不显示过多统计，只保留字数、状态和任务提示。
- 拖动时明确目标位置和跨章影响。

## 8. 右侧上下文栏

固定标签：

```text
本章 | AI | 人物 | 设定 | 检查
```

### 本章

章节目标、SceneBeat、字数、尾快照状态、定稿入口。

### AI

任务类型、模型支持级别、约束摘要、生成参数和任务历史。

### 人物

当前场景人物、当前位置、情绪、持有物、知情状态和声音样本。

### 设定

当前地点、规则、物品、势力与绑定伏笔。

### 检查

当前章问题摘要、风险级别和跳转锚点。

右侧栏仅展示高相关内容；完整编辑跳转到“设定”或“检查”一级入口。

## 9. 全局搜索

入口：顶部栏快捷键和检查工作台。

范围：

- 当前章。
- 全部正文。
- 历史版本。
- 人物与设定。
- 笔记与批注。

默认简单搜索，高级入口才显示正则、大小写和范围组合。

## 10. 弹层、抽屉与独立页面

使用弹层：

- 简短确认。
- 单一对象快速编辑。
- 设置少量参数。

使用抽屉：

- 窄屏右侧上下文。
- AI生成参数。
- 选区操作。
- 搜索结果预览。

使用独立全屏页面：

- 长章节候选比较。
- 三栏冲突处理。
- 导入分章预览。
- 备份恢复。
- 数据损坏诊断。

## 11. 新手与专业模式差异

| 区域 | 新手模式 | 专业模式 |
|---|---|---|
| 一级入口 | 保持六项，次级内容减少 | 全部开放 |
| 规划 | 问题式表单、少字段 | 大纲树与完整字段 |
| 右侧栏 | 摘要和推荐动作 | 全部来源、规则和审计信息 |
| AI | 续写、改写、润色等任务语言 | Run、约束、Prompt版本和模型支持档案 |
| 检查 | 只显示高风险与通俗建议 | 显示规则类型、依据、误报和批量处理 |

模式切换不改变数据、权限和可恢复性。

## 12. 页面跳转与返回

- 从检查问题跳到正文后，返回应回到原问题和滚动位置。
- 从人物摘要进入完整人物页，返回写作时恢复选区和编辑位置。
- 候选比较关闭后回到发起章节和原滚动位置。
- 任务运行中切换页面时，任务条保持可见。
- 应用重启恢复最近项目、章节、编辑位置和侧栏状态；对话框和临时选区不恢复。

## 13. 禁止结构

- 不增加“AI中心”一级入口，AI服务于写作流程。
- 不增加“数据中心”“运营中心”“插件中心”。
- 不把所有功能堆进右侧栏。
- 不让设置页承担正文、候选或项目业务操作。
- 不创建新手版和专业版两套页面代码与数据接口。
