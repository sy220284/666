# WorldForge 新手引导与首次创作规格

> 状态：Frozen  
> 目标：让小白作者快速开始，让资深作者可以跳过流程，避免向导成为强制填表系统。

## 1. 设计原则

1. 五分钟内进入第一章。
2. 所有非数据安全字段都可以跳过或后补。
3. 向导只建立最低可用结构，不要求作者先完成完整设定。
4. 新手模式与专业模式共用数据模型。
5. 资深作者可直接创建空白项目。
6. 帮助在用户需要时出现，每次最多一项主要建议。
7. 跳过步骤后提供轻量脚手架，不用红色错误阻断写作。

## 2. 新建入口

四个并列入口：

| 入口 | 适用 | 结果 |
|---|---|---|
| 快速开始 | 想尽快写第一章 | 最少三步创建项目和第一章 |
| 完整流程 | 想先规划 | 五步向导，仍可跳过字段 |
| 导入已有作品 | 已有TXT/Markdown/DOCX | 进入导入预览 |
| 空白项目 | 熟悉工具或自由写作 | 只填写名称和位置，直接进入编辑器 |

默认推荐“快速开始”，不隐藏其他入口。

## 3. 快速开始流程

### 第1步：项目信息

字段：

- 作品名，必填。
- 项目保存位置，默认本地工作空间，可修改。
- 频道：男频、女频、未指定，可跳过。

页面说明：数据保存在本机；AI需要作者后续自行配置端点。

### 第2步：故事起点

问题式输入：

- 这个故事大概讲什么？
- 主角现在最想得到什么？
- 第一章最想发生什么？

全部可跳过。输入内容映射到ProjectBrief和第一章目标。

### 第3步：开始第一章

选项：

- 直接写。
- 先建立三个场景。
- 让AI给出骨架建议。

AI未配置时第三项显示“需要先配置AI连接”，不阻断其他路径。

## 4. 完整流程

### 步骤1：项目与创作方式

- 项目名称、位置、频道。
- 自主写作、混合创作、AI初稿的介绍。
- 该选择只影响默认入口，可随时改变。

### 步骤2：故事核心

- 一句话高概念。
- 阅读承诺。
- 主角目标。
- 核心冲突。
- 终局方向。
- 必须兑现与禁止内容。

仅作品名必填，其他字段可以稍后补充。

### 步骤3：关键人物

最小人物卡：

- 姓名。
- 身份。
- 当前目标。
- 一句性格或行为边界。

允许只创建一个主角，也允许跳过。

### 步骤4：第一卷与第一章

- 卷名可使用默认。
- 第一章标题可留空。
- 本章目标。
- 预计字数。
- 可选SceneBeat。

### 步骤5：确认与开始

展示将创建的项目结构和本地路径。主操作为“创建并开始写作”。

## 5. 跳过后的轻量脚手架

进入编辑器时根据完成度展示一次性、可关闭提示条。

| 情况 | 提示 |
|---|---|
| 已填写本章目标和SceneBeat | 在右侧“本章”显示目标，不额外提示 |
| 缺少故事核心或人物 | “先用三句话写下这一章想发生什么”，可直接输入并转为SceneBeat |
| 只创建空白项目 | 不显示新手提示，直接空白编辑 |
| 导入作品 | 建议先检查分章和当前章节，不强制补设定 |
| AI路径但未配置Provider | 提供配置入口，同时保留自主写作 |

提示条不遮挡正文，不持续重复出现。

## 6. 新手模式首页

显示：

- 继续写作。
- 当前项目最近章节。
- 建书步骤进度。
- 一条下一步建议。
- 新建或导入。

下一步建议使用确定性规则：

```text
无章节 → 创建第一章
有空章节 → 开始写正文
有正文未保存Version → 可创建历史版本
有定稿无尾快照 → 查看状态变化建议
AI未配置且用户点击AI → 配置连接
```

不使用行为监控或偏好学习推断作者意图。

## 7. 场景化AI引导

AI入口文案按当前上下文变化：

| 当前状态 | 首要建议 |
|---|---|
| 空白章节 | 生成章节骨架 |
| 有SceneBeat无正文 | 按场景扩写 |
| 选中单段 | 改写、精简、扩写 |
| 有完整正文 | 检查衔接或生成备选稿 |
| 候选待处理 | 查看AI建议稿 |
| 定稿完成 | 提取状态变化建议 |

每次最多突出一个建议，其他操作放入菜单。

## 8. 上下文帮助

### 三层帮助

1. 悬停提示：一句话说明功能。
2. 首次使用提示：最多2—3句，带“知道了”和“不再提示”。
3. 页面短帮助：解释当前页面目标和关键边界。

### 术语

新手默认使用：

- 当前稿。
- AI建议稿。
- 历史版本。
- 状态变化建议。
- 检查问题。

专业模式可在帮助中显示对应工程术语。

## 9. 模式切换

入口：设置、命令面板和首页个人偏好区域。

切换效果：

- 字段显隐。
- 默认右侧栏内容。
- 高级信息与工程术语。
- 操作密度。

保持不变：

- 正文。
- 项目结构。
- Candidate、Version和状态。
- 所有权限与安全边界。

切换不要求重启，不弹出警告确认。

## 10. 首次功能提示

只对以下高价值功能提供首次提示：

- 自动保存与本地数据。
- 锁定段落。
- AI建议稿不会自动覆盖当前稿。
- 应用AI建议后可撤销。
- 备份恢复到新副本。
- 沉浸写作。

禁止为每个按钮添加教学弹层。

## 11. 新手错误解释

AI按钮灰色时必须显示原因，例如：

- 尚未配置AI连接。
- 当前选区包含锁定段落。
- 项目处于只读状态。
- 当前AI任务仍在运行。
- 需要先保存当前修改。

不能只用禁用样式而无解释。

## 12. 引导状态存储

只保存：

- 是否完成向导。
- 已看过哪些首次提示。
- 当前模式。
- 是否关闭轻量脚手架。

这些信息保存在本地设置中，不记录“是否认真审核”“停留时长”等行为画像。

## 13. 中断与恢复

- 向导中途退出时，可保存为未完成项目草稿或彻底取消；行为需明确。
- 重新打开后从最近完成步骤继续。
- 创建数据库或目录失败时不留下被误认成正常项目的半成品。
- 导入流程中断回到导入预览，不跳入空白项目。

## 14. 验收标准

- 新用户在无文档帮助下5分钟内进入第一章。
- 空白项目创建不超过2个必填字段：名称、有效保存位置。
- 跳过所有规划步骤仍可写作。
- 新手模式下不显示GenerationRun、EntityState等工程术语作为主文案。
- 专业作者可从启动页直接进入空白编辑器。
- AI不可用时自主写作路径完整可用。
- 首次提示均可关闭，关闭后不重复打扰。
