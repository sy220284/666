# WorldForge V1.0 页面规格

> 状态：Frozen  
> 用途：前端实现和桌面E2E验收依据。

## 1. 启动页

目标：覆盖首次启动、最近项目、Core失败、路径丢失、Schema过新和项目损坏。

```text
顶部：应用名称 / 设置 / 帮助
主体：继续写作 / 新建项目 / 导入已有正文
下方：最近项目
底部：版本 / 本地数据位置 / Core状态
```

主要操作：打开、创建、导入、重新定位路径、移除最近记录、进入设置。

## 2. 新建项目向导

1. 快速开始、完整流程、导入已有或空白项目。
2. 名称、频道和项目位置。
3. 可跳过的ProjectBrief。
4. 可跳过的关键人物。
5. 第一章目标、SceneBeat或直接写作。

完成后创建工作空间、项目库、首卷章和活动Draft。向导未完成时不留下不可识别半成品。

## 3. 首页

新手：继续写作、一步建议、最近项目和新建/导入。  
专业：最近项目、最近章节、运行任务、健康风险和空白项目。

规则：主动提示最多2条；数据风险优先；不展示运营图表。

## 4. 规划工作台

```text
左：卷章与大纲树
中：ProjectBrief / PlotNode / Chapter / SceneBeat编辑
右：人物、设定、伏笔、弧光和目标摘要
```

主要操作：新建卷章、剧情弧和SceneBeat；排序；关联实体和伏笔；跳转正文。

风险：移动有关联正文的SceneBeat先预览；删除规划节点不自动删除正文；规划变化不自动调用AI。

## 5. 写作工作台

```text
顶部：项目/章节、模式、搜索、任务、保存状态
左：卷章目录
中：标题、正文编辑器、状态栏
右：本章 / AI / 人物 / 设定 / 检查
```

正文区：680/760/860px宽度；来源与锁定页边标记；选区工具条；场景分隔。

底部状态：保存、字符数/纯文字字数/目标进度、AI阶段和风险摘要。

主要操作：输入、粘贴、当前章查找、撤销、锁定、快速改写、T0/T1、结构性改写、候选、Version、定稿和沉浸写作。

## 6. 沉浸写作

保留：返回、项目/章节、正文、保存、字数和可关闭轻量AI入口。  
隐藏：一级导航、侧栏、校验详情和非紧急提示。

保存失败和数据风险必须继续显示。退出后恢复原滚动与侧栏状态。

## 7. 右侧栏

### 本章

章节目标、SceneBeat、目标字数、章节状态、尾快照、状态提案和定稿。

### AI

任务、Provider、模型、支持等级、约束摘要、参数、运行任务和历史候选。

### 人物

本章人物、当前状态、知情、声音样本和已确认人物弧光阶段。

### 设定

地点、物品、规则、势力、伏笔和来源。

### 检查

本章问题、弧光一致性、节奏建议、正文锚点和运行检查。

## 8. 候选审阅

顶部：章节、任务类型、候选切换、双栏/上下/单稿/只看差异。  
中部：当前稿、候选稿、结构/字符Diff、SceneBeat导航和采用动作。  
底部：选择数量、冲突数量、应用、保存候选和丢弃。

视口不足时自动从双栏切换上下或单稿。

## 9. 冲突处理

P0默认：当前稿与候选稿双栏，附结果摘要。  
每项显示冲突类型、基线变化、当前、候选、保留当前、采用候选、手动合并或跳过。

所有冲突解决后重新校验Revision并一次事务提交。高级三栏合并结果视图属于P1。

## 10. 历史Version

显示时间、类型、标签、字数、来源Revision、与当前稿比较、恢复为新Draft和导出。

不提供编辑Version按钮。定稿Version默认不可删除。

## 11. 设定工作台

### 实体

人物、地点、势力、道具、能力、规则、事件和自定义实体。

### 中区标签

基础资料、Canon、当前状态、历史状态、人物弧光、关系和引用章节。

### 人物弧光

- 弧光类型、目标、状态和说明。
- 里程碑顺序、关联章节、依赖和`planned/hit/skipped`。
- pending StateProposal单独显示，不能冒充已命中。
- 接受、编辑接受和拒绝通过统一状态提案入口。

### 右区

正文证据、伏笔、知情、冲突和状态提案。

## 12. 时间线、知情与伏笔

时间线：列表/简化轴、起止、精度、人物、地点、依赖和冲突。  
知情：知道、相信、怀疑、误解、未知、来源和获得章节。  
伏笔：生命周期、回收窗口、超期、关联章节和依赖。

V1.0不以复杂自动布局关系图作为主入口。

## 13. 检查工作台

左侧：本章、全书、连续性、设定、人物弧光、文风、机械校对、网文节奏、StoryTodo。  
中部：严重度、标题、章节、锚点、依据和状态。  
右侧：原文、预期依据、建议、跳转、忽略、静音、降级、误报和转待办。

## 14. 网文节奏页

- GenreRhythmProfile开关与参考区间。
- 爽点密度趋势和对应SceneBeat证据。
- 章末钩子建议与结尾锚点。
- 更新目标与实际字数。
- 黄金三章只显示前3章。

所有提示明确标注“建议”，关闭后不再运行或展示。

## 15. StoryTodo与Comment

- StoryTodo按章节、SceneBeat、Block、状态和标签筛选。
- ValidationIssue可转待办。
- 完成待办后重跑来源校验；通过则关闭，未通过则附最新结果。
- Comment提供轻量查看、编辑和删除。

复杂批处理属于P1。

## 16. 搜索与替换

简单模式：查询、范围、结果和上下文。  
高级模式：大小写、正则、实体类型和历史Version。

V1.0 P0范围只搜索Draft、Version和Entity，不显示研究笔记入口。

批量替换：生成ReplacePlan、展示锁定与冲突、创建恢复点、重新校验基线后提交。

## 17. 导入

步骤：文件检查、编码/格式确认、分章预览、合并/拆分/重命名、摘要和提交。

预览期间项目库不变化。取消和失败清理临时数据。

## 18. 导出

选择全书/卷/章、指定Version、TXT/Markdown/DOCX、标题顺序和目标文件。显示真实阶段与结果。

## 19. 备份与恢复

三个区域：日常备份、重大恢复点、手动快照。  
显示时间、类型、大小、验证状态和备注。

恢复流程：选择已验证备份→新目录→验证→注册副本→打开。禁止默认覆盖当前项目。

## 20. 设置

### 通用

语言、启动行为、最近项目和默认模式。

### 编辑器

字体、字号、宽度、行高、段距和首行缩进。打字机模式属于P1，P0不显示可用入口。

### 外观与显示

Theme A/Theme B、对应冻结模式、界面缩放、工作区位置、减少动态和高对比。

### AI连接

Provider列表、地址、模型、凭据引用、连接测试和支持等级。

### 备份

频率、保留数、空间上限和目标目录。

### 检查、节奏与词典

默认检查、静音规则、GenreRhythmProfile和项目词典。

### 快捷键

P0只展示系统默认快捷键；自定义和冲突编辑属于P1。

### 高级

日志、诊断、数据库检查、FTS重建和开发信息。危险操作单独分组。

## 21. 页面通用状态

每页必须覆盖：空、首次加载、刷新、部分失败、完全失败、只读、边界拒绝、运行、取消、冲突和恢复成功。

未实现的P1/V1.5功能不得以可点击占位入口出现。
