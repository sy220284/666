# WorldForge UI专项验收清单

> 状态：Frozen  
> 用途：M4-05与M5发布前UI、交互、高分屏和无障碍验收。每项必须附截图、录屏、测试结果或明确人工记录。

## 1. 文档与实现一致性

- [ ] 一级导航与`INFORMATION_ARCHITECTURE.md`一致。
- [ ] 页面结构与`SCREEN_SPECIFICATIONS.md`一致。
- [ ] 颜色、字体、间距、圆角和动效使用Design Token。
- [ ] 页面内没有散落的未登记颜色和尺寸常量。
- [ ] 新手/专业模式共用数据和页面组件。
- [ ] 沉浸写作作为视图状态实现。
- [ ] 未实现的V1.5页面和入口不以可用状态展示。

## 2. 视觉系统

- [ ] 正文是写作工作台的视觉中心。
- [ ] AI内容未使用大面积蓝色背景。
- [ ] 绿色没有被用于表示AI稿更好。
- [ ] 普通设置和信息列表没有过度卡片化。
- [ ] 候选、冲突、恢复和风险卡片层级清楚。
- [ ] 浅色、深色、护眼和高对比主题完整。
- [ ] SVG图标在100%、125%、150%缩放下清晰。
- [ ] 正文宽度支持680、760、860px。
- [ ] 用户自定义正文字体失败时安全回退。
- [ ] 锁定、来源、警告和危险状态在四种主题下可辨认。

## 3. 启动页与首页

- [ ] 首次启动提供新建、导入和设置入口。
- [ ] 最近项目可打开、定位丢失路径或移除记录。
- [ ] Core启动失败时有明确诊断和重试入口。
- [ ] 新手首页突出继续写作与一条下一步建议。
- [ ] 专业首页显示最近项目与运行任务。
- [ ] 首页主动提示最多2条。
- [ ] 数据安全类提示高于校验和一般提醒。
- [ ] 首页未堆叠运营图表和无关统计。

## 4. 新建项目与引导

- [ ] 四个入口：快速开始、完整流程、导入、空白项目。
- [ ] 空白项目只要求名称和有效保存位置。
- [ ] 非安全字段均可跳过。
- [ ] 五分钟内可进入第一章。
- [ ] 跳过规划后出现可关闭轻量脚手架。
- [ ] 完全空白专业项目不显示新手提示。
- [ ] AI未配置时自主写作仍完整可用。
- [ ] 向导中断不会留下误认成正常项目的半成品。
- [ ] 模式切换不改变项目数据。

## 5. 写作工作台

- [ ] 左卷章目录、中正文、右上下文结构清楚。
- [ ] 右栏标签固定为本章、AI、人物、设定、检查。
- [ ] 正文区长时间阅读舒适。
- [ ] 顶部和底部没有重复堆叠同一状态。
- [ ] 保存、字数、AI任务和检查摘要可见但不抢眼。
- [ ] 章节切换保留滚动位置和编辑位置。
- [ ] 左右侧栏可折叠和调整宽度。
- [ ] 右侧摘要只显示当前章和当前场景高相关信息。
- [ ] 完整编辑可跳转到设定或检查工作台并返回原位置。

## 6. 编辑器

- [ ] 中文拼音输入不丢字、不重复。
- [ ] 五笔或其他组合输入不触发错误快捷键。
- [ ] composition期间不自动保存半成品。
- [ ] 粘贴网页内容清除字体、颜色、脚本和布局。
- [ ] Ctrl/Cmd+Shift+V可纯文本粘贴。
- [ ] 自动保存不阻塞输入。
- [ ] 保存成功只在Core事务确认后展示。
- [ ] 保存失败持续可见并提供重试与复制文本。
- [ ] Enter拆分与Backspace合并符合logicalBlockId规则。
- [ ] 当前章查找替换符合普通编辑器预期。
- [ ] 字数在编辑器、目录、统计和导出预览中一致。

## 7. 锁定与来源

- [ ] 段落可锁定和解锁。
- [ ] 锁定线、图标和悬停说明清楚。
- [ ] 锁定块无法输入、删除、移动和合并。
- [ ] AI、批量替换、拆并章和跨章移动无法修改锁定块。
- [ ] 跳过锁定块时显示非阻断摘要。
- [ ] 默认来源标记低干扰。
- [ ] 来源模式可查看手动、AI、混合和导入来源。
- [ ] 作者修改AI段落后来源更新为mixed。

## 8. 快速改写

- [ ] 单段选区出现浮动工具条。
- [ ] 工具条不遮挡正文和输入法候选窗口。
- [ ] 选区包含锁定内容时AI操作禁用并解释原因。
- [ ] 发起到应用不超过2次主要点击。
- [ ] 应用前原文保持不变。
- [ ] 可换一个、取消和应用。
- [ ] 应用后可立即撤销。
- [ ] Revision变化时进入冲突或重新校验。
- [ ] 跨段与跨SceneBeat操作自动进入结构性候选流程。

## 9. AI任务

- [ ] 阶段对应真实程序事件。
- [ ] 无可计算总量时不显示虚假百分比和倒计时。
- [ ] 显示已运行时间和已接收字符数。
- [ ] 切换章节和页面后任务继续且不串稿。
- [ ] 全局任务条可查看运行任务。
- [ ] 取消后500ms内显示已收到取消。
- [ ] 中断或取消后的部分结果明确标记partial。
- [ ] AI不可用不影响编辑、搜索、备份和导出。
- [ ] 模型支持等级在AI详情中可见。

## 10. 候选审阅

- [ ] 支持双栏、上下、单稿和只看差异。
- [ ] 视口不足时使用上下或单稿。
- [ ] `logicalBlockId`结构Diff正确。
- [ ] 新增、删除、移动、拆分和合并清楚区分。
- [ ] 中文字符差异可读。
- [ ] 场景导航和上一/下一差异可用。
- [ ] 同步滚动可关闭。
- [ ] 未改段可折叠。
- [ ] 块级和SceneBeat级采用可用。
- [ ] 锁定块不可采用。
- [ ] 不使用单一AI分数强制推荐候选。
- [ ] 5000字首屏≤500ms，完整Diff≤1.2s。
- [ ] 20000字候选渐进计算且可取消。

## 11. 冲突与采用回退

- [ ] Revision、Hash、缺失块、锁定和结构冲突分别识别。
- [ ] 冲突不会只显示Toast。
- [ ] 可选择保留当前、采用候选、手动合并或跳过。
- [ ] 提交前再次校验Revision。
- [ ] Candidate采用在一次事务中完成。
- [ ] 应用后定位首个修改块。
- [ ] 显示“撤销本次应用”。
- [ ] Ctrl/Cmd+Z可整体撤销。
- [ ] 重启后可通过Checkpoint恢复采用前状态。
- [ ] 采用后继续编辑产生冲突时使用回退预览。

## 12. 规划与设定

- [ ] 大纲树可键盘和拖拽操作。
- [ ] 拖动显示准确目标位置。
- [ ] 规划变化不自动修改正文。
- [ ] 删除SceneBeat不删除正文。
- [ ] 跨章移动有关联正文时显示影响预览。
- [ ] Canon、当前状态和历史状态视觉分离。
- [ ] AI状态提案与已确认状态视觉分离。
- [ ] 时间线、知情和伏笔均有列表式可用入口。
- [ ] 关系图和时间轴不是唯一操作方式。

## 13. 检查、搜索与替换

- [ ] 确定性、统计和AI语义问题类型可区分。
- [ ] AI语义问题使用风险文案。
- [ ] 问题有正文锚点、依据和建议。
- [ ] 可忽略、静音、降级、标记误报和转待办。
- [ ] 从问题跳正文并可返回原位置。
- [ ] 全项目搜索结果包含范围和来源。
- [ ] 正则放在高级入口。
- [ ] 批量替换先预览。
- [ ] 锁定块默认跳过。
- [ ] ReplacePlan过期时阻止提交。
- [ ] 高风险替换前创建恢复点。

## 14. 导入导出

- [ ] TXT编码置信度不足时允许人工选择。
- [ ] 导入先预览分章、合并、拆分和重命名。
- [ ] 预览期间项目数据库不改变。
- [ ] DOCX异常文件有安全错误提示。
- [ ] 导入取消和失败后临时数据清理。
- [ ] 导出明确选择Version。
- [ ] 导出预览显示范围、顺序和字数。
- [ ] 目标文件冲突有明确决策。
- [ ] 导出完成提供打开文件或文件夹入口。

## 15. 备份与恢复

- [ ] 日常、重大操作和手动快照分组清楚。
- [ ] 每个备份显示验证状态、时间、大小和备注。
- [ ] 最后一份已验证备份不能删除。
- [ ] 数据损坏时可进入恢复页。
- [ ] 恢复默认选择新目录。
- [ ] 原项目不会被直接覆盖。
- [ ] 恢复后注册新项目并通过完整性检查。
- [ ] 空间不足和清理策略有明确提示。

## 16. 交互状态

- [ ] 所有核心页面有空状态。
- [ ] 短加载不滥用全屏转圈。
- [ ] 长任务显示阶段和取消。
- [ ] 用户取消不显示红色错误。
- [ ] 保存失败、只读和损坏使用持续Banner。
- [ ] 数据冲突使用专门视图。
- [ ] 危险确认说明对象、影响和恢复能力。
- [ ] 高频成功反馈不持续干扰。
- [ ] Toast消失后重要结果仍可找到。

## 17. 响应式与DPI

### 1280×800

- [ ] 可完成新建、写作、AI、候选、定稿和导出。
- [ ] 右侧栏使用抽屉。
- [ ] 候选默认上下或单稿。
- [ ] 无整页横向滚动。

### 2560×1440

- [ ] 100%、125%、150%缩放均清晰。
- [ ] 125%下完整三栏可用。
- [ ] Popover、菜单和对话框不越界。
- [ ] 正文行宽保持受控。

### 21:9

- [ ] 工作区支持居中、偏左和偏右。
- [ ] 最大工作区宽度受控。
- [ ] 高频和危险操作靠近正文。
- [ ] 物理屏幕边缘不放长文本与精确操作。

### 混合DPI

- [ ] 窗口跨屏后布局重新计算。
- [ ] DIP坐标恢复正确。
- [ ] 选区工具和弹层重新定位。
- [ ] 未保存文本和选区不丢失。

## 18. 无障碍

- [ ] 仅键盘完成核心创作流程。
- [ ] 焦点顺序与视觉顺序一致。
- [ ] 对话框与抽屉限制焦点并正确返回。
- [ ] 危险操作默认焦点不在危险按钮。
- [ ] 所有图标按钮有可访问名称。
- [ ] 锁定、冲突和状态不只靠颜色。
- [ ] 保存失败和只读状态可被读屏感知。
- [ ] Candidate Diff有结构化差异列表。
- [ ] 高对比与减少动效可用。
- [ ] 200%缩放下核心操作可达。

## 19. 性能与稳定性

- [ ] 2K键入P95≤50ms。
- [ ] 自动保存P95≤150ms。
- [ ] 编辑IPC P95≤200ms。
- [ ] 正文滚动≥50fps。
- [ ] 侧栏开关不触发整章重渲染。
- [ ] 主题和缩放切换不丢选区和文本。
- [ ] Core断开时编辑器停止虚假保存并允许复制文本。
- [ ] 长章节降级只关闭派生装饰，不关闭正文安全功能。

## 21. 阻断发布的问题

- 保存状态与实际事务结果不一致。
- 锁定块存在任何可修改路径。
- 未确认Candidate进入当前稿。
- Revision冲突静默覆盖。
- 1280×800无法完成核心流程。
- 2K 125%文字或图标明显模糊。
- 中文输入法造成丢字或误操作。
- 保存失败、数据库损坏或只读状态不可见。
- 核心流程无法仅用键盘完成。
