# WorldForge UI专项验收清单

> 状态：Frozen  
> 用途：`M7-01`—`M7-03`与M8发布前UI、交互、高分屏和无障碍验收。  
> 证据：截图、录屏、桌面E2E、键盘记录或明确人工验收。

## 1. 文档与实现一致

- [ ] 一级导航与信息架构一致。
- [ ] 页面结构与屏幕规格一致。
- [ ] Design Token统一，业务组件无散落颜色和尺寸。
- [ ] 未实现功能不显示可用入口。
- [ ] 新手/专业模式共用数据、命令和页面组件。
- [ ] Theme A/Theme B不分叉业务逻辑。

## 2. 基础写作MVP

- [ ] AI未配置时可新建项目、建卷章、写作、保存、版本、导入导出和恢复。
- [ ] 最近项目可打开、重新定位或移除。
- [ ] Core启动失败有诊断和重试。
- [ ] 中文输入不丢字、不重复。
- [ ] composition期间不保存半成品。
- [ ] 粘贴清除脚本、字体、颜色和布局污染。
- [ ] 自动保存不阻塞输入。
- [ ] 保存成功只在Core事务确认后展示。
- [ ] 保存失败持续可见，可重试和复制未保存文本。
- [ ] 字数在编辑器、目录、统计和导出预览一致。
- [ ] 当前章查找符合普通编辑器预期。
- [ ] Version只读，恢复产生新当前稿。
- [ ] TXT/Markdown导入导出可用。
- [ ] 损坏项目可只读浏览和恢复副本。

## 3. 写作工作台

- [ ] 左卷章、中正文、右上下文结构清楚。
- [ ] 正文为视觉中心。
- [ ] 保存、字数、任务和检查摘要可见但不抢眼。
- [ ] 章节切换保留编辑和滚动位置。
- [ ] 侧栏可折叠。
- [ ] 沉浸写作是视图状态。
- [ ] 跳转其他工作台后可返回原位置。

## 4. 锁定、Patch与版本

- [ ] 段落可锁定/解锁。
- [ ] 锁定块无法输入、删除、移动、拆分和合并。
- [ ] AI、替换、拆并章不能修改锁定块。
- [ ] Revision/Hash冲突进入专门视图。
- [ ] Candidate采用一次事务完成。
- [ ] 应用后可整体撤销。
- [ ] 重启后可恢复采用前状态。
- [ ] Version和历史正文不可直接修改。

## 5. 规划与连续性

- [ ] 大纲树支持键盘和拖拽。
- [ ] 规划变化不自动改正文。
- [ ] 删除SceneBeat不删除正文。
- [ ] 跨章移动有关联正文时显示影响预览。
- [ ] Canon、当前状态、历史状态和AI提案视觉分离。
- [ ] 时间线、知情、伏笔和人物弧光有列表式入口。
- [ ] pending状态提案不显示为权威状态。
- [ ] 返修旧章后stale状态清楚但不自动改后文。

## 6. AI任务与候选

- [ ] AI阶段对应真实程序事件。
- [ ] 无总量时不显示虚假百分比和倒计时。
- [ ] 切章/切页后任务继续且不串稿。
- [ ] 取消后500ms内反馈。
- [ ] 中断/取消的部分结果明确标记partial。
- [ ] AI不可用不影响基础写作、搜索、备份和导出。
- [ ] 模型支持等级可见。
- [ ] T0可绕过。
- [ ] T1临时流不直接写Draft。
- [ ] 快速改写应用前原文不变。
- [ ] 跨段操作自动进入结构性Candidate。
- [ ] 融合结果显示节拍来源。

## 7. 候选审阅

- [ ] 支持双栏、上下、单稿和只看差异。
- [ ] 视口不足时自动切换布局。
- [ ] 结构Diff和中文字符Diff可读。
- [ ] 新增、删除、移动、拆分和合并区分清楚。
- [ ] 支持整稿、块级和SceneBeat级采用。
- [ ] 锁定块不可采用。
- [ ] 冲突可保留当前、采用候选、手动合并或跳过。
- [ ] 不使用单一AI分数强制推荐。
- [ ] 5000字Diff首屏≤500ms、完整≤1.2s。
- [ ] 20000字渐进计算可取消。

## 8. 校验、搜索与节奏

- [ ] 确定性、统计和AI语义问题可区分。
- [ ] AI问题使用风险文案和证据。
- [ ] 可忽略、静音、降级、误报和转待办。
- [ ] 全项目搜索显示范围和来源。
- [ ] 批量替换先预览、检查锁定并创建恢复点。
- [ ] ReplacePlan过期阻止提交。
- [ ] 节奏指标为建议级、可关闭、阈值可编辑。
- [ ] 黄金三章只作用于前3章。

## 9. 导入导出与恢复

- [ ] TXT编码低置信度可人工选择。
- [ ] 导入先预览分章、合并、拆分和重命名。
- [ ] 预览期间项目数据库不变化。
- [ ] DOCX异常和恶意文件有安全错误。
- [ ] 取消/失败后临时数据清理。
- [ ] 导出明确选择Version。
- [ ] 日常、重大操作和手动快照分组清楚。
- [ ] 备份显示验证状态、时间、大小和备注。
- [ ] 最后一份已验证备份不能删除。
- [ ] 恢复默认到新目录，原项目不覆盖。

## 10. 状态与帮助

- [ ] 核心页面有空状态。
- [ ] 长任务显示阶段和取消。
- [ ] 用户取消不显示红色错误。
- [ ] 保存失败、只读和损坏使用持续Banner。
- [ ] 数据冲突使用专门视图。
- [ ] 首页主动提示最多2条。
- [ ] P0数据安全高于P1任务、P2决策和P3信息。
- [ ] 悬停、首次提示和页面帮助有统一Registry。

## 11. 视觉主题

- [ ] Theme A浅色、深色、护眼和高对比完整。
- [ ] Theme B浅色、深色和印章表现层完整。
- [ ] 主题切换不改变业务结果。
- [ ] AI来源不用绿色表示更优。
- [ ] 印章只在命令成功后播放。
- [ ] 减少动态时所有功能可用。
- [ ] 锁定、来源、警告和危险状态在冻结主题范围内可辨认。
- [ ] SVG图标在100%、125%、150%缩放清晰。

## 12. 响应式与DPI

### 1280×800

- [ ] 可完成新建、写作、AI、候选、定稿和导出。
- [ ] 侧栏按规格使用抽屉。
- [ ] 无整页横向滚动。

### 2560×1440

- [ ] 100%、125%、150%均清晰。
- [ ] 125%下完整工作台可用。
- [ ] Popover、菜单和对话框不越界。

### 21:9与混合DPI

- [ ] 工作区支持居中、偏左和偏右。
- [ ] 正文行宽受控，主要操作不漂远。
- [ ] 跨显示器窗口坐标和浮层重定位正确。
- [ ] 显示器移除后窗口仍可见。

## 13. 无障碍

- [ ] 核心流程可全键盘完成。
- [ ] 焦点顺序、焦点环和对话框焦点陷阱正确。
- [ ] 图标有可访问名称。
- [ ] 状态不只靠颜色表达。
- [ ] 屏幕阅读器可读保存、任务、冲突和恢复状态。
- [ ] 自定义字体失败安全回退。

## 14. 发布阻断

以下任一项失败，M7/M8不得Verified：

- 基础写作必须依赖AI。
- 保存状态与Core事务不一致。
- 主题分叉业务逻辑。
- 锁定、Revision、Candidate或Version硬保证被绕过。
- 1280×800无法完成核心流程。
- 恢复覆盖原项目。
- 未实现入口伪装可用。
