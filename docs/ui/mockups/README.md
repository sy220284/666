总览包含以下 8 个核心界面：

1. 写作工作台
2. 首页 / 项目总览
3. 规划工作台
4. Candidate 候选审阅
5. 连续性工作台
6. 检查与交付工作台
7. 设置中心
8. AI 配置与本地服务管理
